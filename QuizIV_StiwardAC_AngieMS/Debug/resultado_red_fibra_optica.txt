ALGORITMO UTILIZADO: Kruskal

El algoritmo de Kruskal encuentra el árbol de expansión
mínima (MST) mediante los siguientes pasos:

1. Ordenar todas las aristas por peso (menor a mayor)
2. Inicializar cada nodo como un conjunto independiente
3. Para cada arista en orden:
   - Si conecta dos conjuntos diferentes: seleccionarla
   - Si crea un ciclo: descartarla
4. Continuar hasta conectar todos los nodos
